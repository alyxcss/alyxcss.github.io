# 实验报告：GitHub个人博客搭建
## 实验目的
使用 GitHub Pages 快速搭建个人博客，并将本报告作为第一篇博文发布。

## 实验工具
GitHub 账号（仅浏览器，无需安装任何软件）

## 实验步骤
1. 创建 GitHub 仓库：用户名.github.io
2. 开启 GitHub Pages 功能
3. 编写 Markdown 实验报告并提交
4. 访问 https://你的用户名.github.io 查看结果

## 实验结果
博客已成功上线！本实验报告即为第一篇博文。

## 总结
GitHub Pages 是最简单、免费的个人网站搭建方式，纯浏览器操作，无需服务器。
